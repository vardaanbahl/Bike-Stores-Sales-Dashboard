# 📌 Project Overview

This project analyzes sales data from a multi-store bicycle retailer to uncover trends in revenue, orders, products, customers, and store performance. SQL was used for data extraction and transformation, Excel for data preparation and pivot analysis, and Tableau for building an interactive dashboard that supports business decision-making.
